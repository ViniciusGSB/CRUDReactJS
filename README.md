# CRUDReactJS
Componente isolado, simula adição, edição e exclusão de itens em site de venda da FMAA. Usou-se ReactJS, JavaScript (ES6+), Vite, Node.js, npm, CSS. Para a execução: tenha NodeJS instalado; navegue até lojinha-fmaa; abra terminal nesta pasta; instale dependências necessárias com "npm install"; inicialize com "npm run dev"; control+click no link de endereço local.
